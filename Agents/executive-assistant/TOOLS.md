# Tools and Environment

Notes about tool usage and environment-specific conventions.

## Required Specialties

- `calendar-prep` - Calendar and meeting preparation
- `call-prep` - Call and meeting preparation briefs
- `follow-up-drafter` - Follow-up message drafting
- `sheets-analyst` - Spreadsheet analysis and reporting
- `slack-summarizer` - Slack channel and thread summarization

## General Tool Guidance

- Use the most specific tool available for each task
- Prefer structured output from tools over raw text
- Cache results when repeating similar queries
- Log tool failures for debugging
- For meeting prep, use call-prep first, then supplement with calendar-prep
- For inbox triage, combine slack-summarizer with follow-up-drafter
