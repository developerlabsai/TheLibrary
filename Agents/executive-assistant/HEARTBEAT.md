# Heartbeat Tasks

These tasks run on a periodic schedule. Keep this file concise to preserve tokens.

## Daily Checks

1. Review calendar for upcoming meetings in the next 24 hours
2. Check for pending follow-ups that are overdue or due today
3. Summarize unread messages in priority channels
4. Identify any scheduling conflicts or overbooking
5. Prepare daily brief with top 3 priorities

## Weekly Checks

1. Review open loops from the past week
2. Prepare weekly review (wins, blockers, upcoming deadlines)
3. Check for meetings without agendas
4. Identify delegations that need follow-up

## Rules

- If nothing needs attention, return HEARTBEAT_OK
- Only surface items that require action
- Keep heartbeat responses brief
