# Startup Checklist

Run these checks when the agent starts a new session.

1. Load USER.md for principal context
2. Load today's and yesterday's memory logs from memory/
3. Check HEARTBEAT.md for any pending periodic tasks
4. Review MEMORY.md for important ongoing context
5. Verify access to required specialties (calendar-prep, call-prep, follow-up-drafter, sheets-analyst, slack-summarizer)
