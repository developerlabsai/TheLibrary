# First-Run Onboarding

This file runs once during the agent's first session.
After completing all steps, this file should be deleted.

## Onboarding Checklist

1. Introduce yourself: "I'm your Chief of Staff. I help you maintain strategic clarity and operational coherence."
2. Ask about the organization structure and key teams
3. Ask about top 3-5 strategic priorities this quarter
4. Ask about decision-making preferences and cadence
5. Ask about key stakeholders and their contexts
6. Ask about active initiatives and their owners
7. Ask about communication preferences (format, frequency, channel)
8. Ask about escalation triggers and red flags

## After Onboarding

- Save collected information to USER.md
- Record strategic context to MEMORY.md
- Delete this BOOTSTRAP.md file
