Brand Configuration Reference

7 preset themes + custom branding for the entire Dev Labs template system

How to Brand Any Template

  1. Add the brand-config.css stylesheet link after the template's <style> block
  2. Add a data-brand attribute to the <html> tag
  3. Done. Every component in the template inherits the new brand colors.
<html lang="en" data-brand="midnight">
<head>
  <style>
    /* ... template's original styles ... */
  </style>
  <!-- Brand override AFTER the style block -->
  <link rel="stylesheet" href="brand-config.css">
</head>

The brand-config.css file contains a variable bridge that maps generic --brand-* names to each template family's native variable names (--violet, --navy, --gold, --cyan, etc.). This means one brand preset works across all 8 templates without any template modifications.

Creating a Custom Brand

For Client-Specific Branding

Add a new block to brand-config.css with the client's brand colors. You need 5 core values — everything else derives from them.

/* Minimum required: 5 colors */
[data-brand="acme-corp"] {
  /* 1. Primary brand color (hex from client brand guide) */
  --brand-primary: #2E5BFF;
  --brand-primary-lt: #6C8CFF;   /* 30% lighter */
  --brand-primary-bg: #EEF2FF;   /* 95% lighter */
  --brand-primary-bdr: #D6DEFF;  /* 85% lighter */

  /* 2. Secondary accent (complement or analogous) */
  --brand-secondary: #00C48C;
  --brand-secondary-bg: #E6FFF6;
  --brand-secondary-bdr: #B3FFE0;

  /* 3. Sidebar (usually darkest brand color or near-black) */
  --brand-sidebar: #0A1628;
  --brand-sidebar-hov: #152238;

  /* 4. Surface (page background — keep very light) */
  --brand-surface: #F8FAFF;
  --brand-card: #FFFFFF;

  /* 5. Text scale */
  --brand-text: #0A1628;
  --brand-text-mid: #4B5C7E;
  --brand-text-muted: #8E9BB7;
  --brand-border: #DDE3EF;
  --brand-border-lt: #EEF1F7;

  /* Semantic colors (usually keep defaults) */
  --brand-success: #16a34a;
  --brand-success-bg: #f0fdf4;
  --brand-warn: #d97706;
  --brand-warn-bg: #fffbeb;
  --brand-danger: #e11d48;
  --brand-danger-bg: #fff1f2;
  --brand-info: #2563eb;
  --brand-info-bg: #eff6ff;
}

Quick method: Take the client's primary brand hex, paste it into uicolors.app, and use the generated 50/100/500/600 shades for bg/border/light/primary. Use the 900 shade for sidebar and text.
