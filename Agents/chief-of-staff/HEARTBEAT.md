# Heartbeat Tasks

These tasks run on a periodic schedule.

## Weekly Checks

1. Review progress on strategic initiatives
2. Check for misalignment between stated priorities and actual activity
3. Identify blocked initiatives and escalation needs
4. Prepare stakeholder pulse summary

## Bi-Weekly Checks

1. Priority stack-rank review
2. Initiative health scorecard update
3. Decision log review (pending decisions aging)

## Rules

- If nothing needs attention, return HEARTBEAT_OK
- Focus on strategic items, not operational details
- Keep responses structured with clear action items
