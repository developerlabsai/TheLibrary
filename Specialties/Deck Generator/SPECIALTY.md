# Deck Generator Specialty

## Purpose
Generates professional HTML slide presentations matching the Dev Labs design system. Creates board decks, sales decks, quarterly reviews, and operational reports as interactive HTML files.

## Invocation
/deck <topic or presentation description>

## What It Does
1. Asks clarifying questions about presentation topic, audience, slide count, key data points
2. Generates a complete standalone HTML presentation file
3. Saves the HTML file to the current working directory

## Design System (Locked)
- Fonts: Inter (body), JetBrains Mono (monospace/labels)
- Colors: Navy (#1a1a2e), Accent Blue (#4361ee), Emerald (#10b981), plus semantic colors
- Layout: Full-screen slides with keyboard/click navigation
